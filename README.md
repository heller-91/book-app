# Book List App

A simple React application to view books and add new books using a popup form.

## Installation

```bash
npm install
```

## Run the App

```bash
npm start
```

The app will open at:

http://localhost:3000

## Run Tests

```bash
npm test
```

## How to Add a New Book

1. Click the **New Item** button at the top right of the page.  
2. A popup form will appear.  
3. Enter the **Title** (required).  
4. Optionally add a description and an image URL.  
5. Click **Add Book**.  
6. The new book will appear at the top of the list.
